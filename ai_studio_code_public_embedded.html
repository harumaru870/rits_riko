Syllabus Planner

シラバスから、迷わず時間割を組む

公開用データを最初から内蔵した単一HTMLです。ページを開くとすぐ授業一覧が出て、曜日・時限・キーワードで絞り込みながら右側のマイ時間割へ追加できます。

公開版ではCSVを埋め込み済みです。必要があればこの欄から別のCSVに差し替えられます。Excel書き出しは `Shift_JIS` が合うことがあります。
